Introduction to JavaScript Functions

Master the fundamentals of JavaScript functions, including declarations, expressions, arrow functions, and scope. Build interactive web applications with confidence.

25 minutes Beginner 15 XP
functions.js 
// Function Declaration
function greetUser(name) {
  return `Hello, ${name}! Welcome to CodeQuest.`;
}

// Arrow Function
const calculateArea = (width, height) => {
  return width * height;
};

// Function Call
const message = greetUser("Developer");
const area = calculateArea(10, 5);

console.log(message); // Hello, Developer!
console.log(area);    // 50
